Quais são as possíveis análises de dados em vendas que podem ser feitas? Para essa pergunta, algumas variáveis como os dados disponíveis, o tipo e problema de negócio a ser resolvido, devem também ser consideradas. Mas de modo simplista, as seguintes análises podem ser feitas:
-Qual o valor total vendido?
-Quantas vendas foram realizadas por categoria de produto?
-Quantas vendas foram realizadas por região considerando a prioridade de entrega?
-Qual foi a média de desconto nas vendas por subcategoria de produto?
-Quais países tiveram maior média de valor de venda?
